# MatchMaker — 组织者快速上手指南

**Languages**: [English](ORGANIZER_GUIDE.md) · 简体中文

🔗 **App 地址**：**<https://CaiZhou2.github.io/MatchMaker/>**

你正在组织一个每周的小型比赛，有人把 MatchMaker 分享给你了。这份指南能让你在 10 分钟内从零开始办完第一场比赛 —— 不需要写代码，不需要注册账号。

> **一句话说明**：用手机浏览器打开上面那个 URL → 添加到主屏幕 → 开始第一场比赛。**每场结束后一定要导出备份。** 就这么简单。

---

## 这是个什么东西？

MatchMaker 是一个跑在手机浏览器里的小工具（不需要应用商店、不需要登录、比赛过程中不需要联网）。它做四件事：

1. **管理选手名单**，跨周累积积分和胜率
2. **自动平衡组队**，按每个人的胜率抽队长 + 随机分配队友
3. **自动选赛制**，根据时间和场地数挑小组赛+淘汰赛、循环赛或者随机配对
4. **比赛中录入结果**，结束时一键写入所有人的积分和消费

所有数据**只存在你这台手机上**，不会上传到任何服务器，其他组织者看不到你的数据。

---

## 第 1 步：装到手机上（一次性，约 2 分钟）

这是一个 PWA（渐进式 Web 应用），可以从一个 URL 像普通 App 一样装到主屏幕 —— 不需要 App Store / 应用市场。

### iOS（iPhone / iPad）

1. **必须用 Safari** 打开 MatchMaker 的 URL（不能用 Chrome —— iOS 只允许 Safari 装 PWA）。
2. 点底部的**分享**按钮（带向上箭头的方框图标）。
3. 在分享菜单里向下滚，点**添加到主屏幕**。
4. 右上角点**添加**。
5. 主屏幕出现一个 🏆 图标。点开它，App 全屏打开，没有浏览器外壳。

### Android

1. 用 Chrome（或 Edge / 三星浏览器 / 任何现代浏览器）打开 MatchMaker 的 URL。
2. 点右上角 **⋮** 菜单。
3. 点**安装应用**（或**添加到主屏幕**）。
4. 确认安装对话框。
5. 应用抽屉和主屏幕都会出现 🏆 图标。

### 验证装好了

- 完全关闭 Safari / Chrome
- 点主屏幕的图标
- 应用应该全屏打开（看不到 URL 栏）
- 试试**打开飞行模式**再打开应用 —— 应该还能用。能用就说明装好了，**完全离线可用**。

---

## 第 2 步：建立选手名单（一次性）

这一步只需要做一次。以后新加入的人随时往里加就行。

1. 打开 App → 点 **📇 选手数据库**
2. 输入姓名，点**添加**。把你这一组的所有人都加一遍。
3. 列表会显示每个人的积分、胜率、参赛周数和累计消费。第一次都是零。

> 💡 不一定要一次性把所有人都加进去。在比赛设置页面有个**快速添加**输入框，可以在新人第一次出现的时候直接添加。

---

## 第 3 步：办第一场比赛（设置 5 分钟，然后开打）

1. 主页 → **开始本周比赛**
2. **勾选今天到场的人**，顶部会显示已选人数
3. 设置 4 个参数：
   - **每队人数** —— 双打 2，单打 1
   - **场地数** —— 你今天有几块场地
   - **每场时长（分钟）** —— 羽毛球一般 15 分钟
   - **总时间（分钟）** —— 场馆订到几点
4. **本周消费** —— 场馆费用（人民币或别的，App 不在乎单位）。完成比赛时会自动均摊到每个参赛者。
5. 点**生成队伍**

### 你会看到什么

- **队伍** —— 自动生成的阵容。👑 标记是队长（每队胜率最高的那个）。每队 1 个队长 + 随机分配的队友，所以队伍之间实力大致平衡。
- **推荐赛制** —— 队伍 ≥4 且时间够时选小组赛+淘汰赛；队少或时间紧选循环赛；连循环赛都装不下时切到"随机配对（公平模式）"
- **🔄 重新随机** —— 重新抽一次
- **🔀 手动调整** —— 点两个选手互换位置
- **📋 复制比赛安排** —— 把赛程复制成纯文本，粘贴到群里（微信群、QQ 群、Telegram 都行），所有人都能看

确认阵容满意后，点**开始比赛**。

---

## 第 4 步：比赛中录入结果

比赛页面按时间段显示所有比赛。每场比赛：

- **A胜 / 平局 / B胜** 三个按钮，一点就记录
- 也可以在两个数字框里直接输入比分（比如 `21` 和 `15`），结果会**自动**根据比分判断（高的赢，相等是平局）
- 再点同一个结果按钮可以清除（误点了用）

顶部进度条显示 `已完成 / 总场次`。所有比赛都录入后，**完成比赛 & 更新积分** 按钮才会出现。

> 💡 录入比分是可选的。如果今天打得快，只用三个按钮就够了。比分的好处是：小组赛积分相同时用净胜分作为决胜规则，历史详情里也好看一些。

---

## 第 5 步：完成比赛（最重要的一步）

点**完成比赛 & 更新积分** → 确认。

App 会自动做三件事：

1. **把所有人的新积分 / 胜负 / 消费写入数据库**。提交之后这一步在 App 里就撤销不了了。
2. **自动下载一份备份文件**到你手机的下载文件夹，文件名是 `matchmaker-backup-YYYY-MM-DD.json`。成功页面顶部会出现绿色的"💾 备份已自动下载"提示。
3. **显示一个 📤 分享备份 按钮**（如果你的手机支持的话）。点一下就能用微信 / 邮件 / AirDrop 把备份文件发出去。

### ⚠️ 这一段一定要看 —— 决定了你以后会不会哭

**每场比赛后都把自动下载的备份发到一个安全的地方。** 最简单的流程：

- 点 **📤 分享备份** → 选 **微信** → 选 **文件传输助手**
- 完事。备份现在在云端了，任何设备都能拿到，免费，永久。
- 或者：发到自己邮箱、AirDrop 到笔记本、保存到 iCloud Drive、夸克网盘、百度网盘 …… 都行。

**为什么这件事这么重要：**

- iOS Safari 有一个叫 ITP 的"智能"隐私功能，**7 天没打开 App 就可能清掉所有数据**。如果你跨过一周（节假日、出差），可能下次打开发现一切归零。
- 换手机 / 清理浏览器 / 浏览器抽风都可能让数据全没。
- App 内部有保护层（数据会镜像备份到一个更耐用的存储里），但**只有你聊天记录里的那个文件是 100% 可靠的备份**。

**最坏情况发生时**（打开 App 发现选手清零了）：看下面的"故障排查"。

## 第 6 步：日常查看数据

主页有三个排行榜 Tab：

- **积分榜** —— 累计积分（胜 = 3 / 平 = 1 / 负 = 0）
- **胜率榜** —— 胜场 ÷ 总场次
- **参与榜** —— 参与周数最多（适合评"年度勤勤恳恳奖"之类的）

在**📇 选手数据库**里点任意选手，或者用**🔍 选手查询**搜，可以看到选手详情页：完整的统计数据 + 对战每个对手的"对位胜率"表格。

主页的消费统计卡片显示累计总消费。**复制消费信息**会把每个人的消费明细复制成文本，发到群里（"这是大家本月的费用"）。**清零消费**重置所有人的累计消费（比如月初清零）—— 在下次比赛提交前可以一键恢复。

## 软件更新

开发者推送新版本时，App 顶部会弹出 **"✨ 有新版本可用 / 立即更新"** 横幅。点**立即更新**，App 就会重新加载到新版本。**你的数据在更新中不会丢**。

如果你觉得应该有新版本但没看到横幅：把 App 完全关闭（从最近任务里左滑或上滑划掉），再重新打开，横幅就会出现。

---

## 换手机 / 数据丢了之后导入

1. 把备份的 JSON 文件传到新手机（比如从微信文件传输助手下载）
2. 在新手机上打开 MatchMaker
3. 点 **📇 选手数据库** → 滚到**数据导入 / 导出** → 点 **📥 导入 JSON**
4. 选择那个备份文件。App 会显示 "X 个选手, Y 条历史 — 确认？" → 确认
5. 完成。所有数据回来了。

## 故障排查

**App 显示零选手，但我昨天还有数据。**
- iOS Safari 可能因为 ITP 7 天清理策略清掉了 localStorage
- App 的 IndexedDB 影子备份**应该**会在下次打开时自动恢复数据。如果是的话，你只会看到选手们悄悄出现
- 如果没自动恢复，导入你最新的 JSON 备份（看上面"换手机"那一节）

**iOS 上没法添加到主屏幕。**
- 必须用 Safari，不能用 Chrome 或别的浏览器。iOS 只允许 Safari 装 PWA。

**完成比赛页面没有 📤 分享备份 按钮。**
- 你的浏览器不支持 Web Share API 文件分享。但 App 应该已经把备份自动下载到下载文件夹了 —— 从那里手动分享就行。

**录入比分后没有自动选结果。**
- 两个比分都要填。两个数字都填了之后，结果会自动判断（高的赢，相等是平局）。

**点错了结果按钮。**
- 点正确的那个，新值会覆盖旧值。（再点同一个按钮会**清除**结果 —— 也可以用这个方法撤销重输。）

**我要改选手姓名 / 合并两个选手。**
- 目前只能用：导出 → 手工编辑 JSON 文件 → 导入。App 还没有内建的改名和合并功能。

**比赛装不下我的时间预算。**
- App 会自动切换到"随机配对"模式，会有横幅说明。如果连这个都不行，只剩这几个选项：加时间、加场地、减人数、缩短每场时长。
